# Zaim Lens

Zaim Lens は、AIによる高度な購入履歴のスクリーンショット・レシート解析と、アカウント間の履歴同期を統合した、Zaimユーザーのための次世代入力・同期ハブです。

**App URL:** [https://zaim-lens-472419292018.asia-northeast1.run.app](https://zaim-lens-472419292018.asia-northeast1.run.app)

**【免責事項】**  
本ウェブサービスおよびプログラムは個人が開発した非公式のものであり、株式会社くふうカンパニーおよび「くふう Zaim」とは一切関係ありません。

## 🌟 主な機能

### 1. レシート解析 (Receipt Parser)
* あらゆるソースに対応: ネットショップの購入履歴スクショから、紙のレシート画像まで、AI (Gemini) が読み取ります。
* 爆速バッチ処理: 複数枚のレシートを一度にアップロードし、裏側で連続解析。次々と確認・登録が可能です。
* 手入力モード: レシートがない端数支払いや、解析なしでのクイックな登録にも対応しています。

### 2. 履歴コピー (History Copy)
* アカウント間連携: 「個人用」のアカウントから「共有用」のアカウントへ、必要な明細だけを選んで一括コピー。家族間の精算や管理を効率化します。
* 柔軟な期間指定: 過去の履歴を自由な期間（月指定、カレンダー範囲指定）で取得し、カテゴリを修正しながらコピー先へ反映できます。

### 3. クラウドネイティブなアカウント管理 (OAuth 連携)
* 複数アカウント運用: 家族用と個人用など、複数のZaimアカウントをOAuth 1.0aで安全に連携。トークンはFirestoreに暗号化保存されます。
* UIからの直接設定: 開発者コンソールでの複雑なアクセストークン発行は不要。アプリ内の「Zaim連携設定」から簡単に連携可能です。

### 4. 安心・安全の設計 (Robust Design)
* 重複登録防止: Zaim側に同じ日付・金額の明細がないかAIが事前にチェック。二重計上のミスを未然に防ぎます (32-bit安全な厳密なID管理を実施)。
* App Shell アーキテクチャ: 初期ロード時のチラつきを防止し、ネイティブアプリのようなシームレスな起動体験を提供します。

## 📱 使い方 (PWAの活用)

本アプリは **Progressive Web App (PWA)** です。ホーム画面に追加することで、ダークモードに対応したネイティブアプリさながらの体験を提供します。

1. アプリのURLにアクセスします。
2. ブラウザのメニュー（iOSなら共有ボタン、Androidなら3点リーダー）から **「ホーム画面に追加」** を選択します。
3. ホーム画面のアイコンから起動してください。
